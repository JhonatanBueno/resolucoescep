R E S O L U Ç Ã O  N°  181/85 - CEP

 

Altera Fluxograma do Currículo do Curso de Administração.

 

Considerando o contido nas folhas 189 a 195 do Processo nº 0982/77;

 

O CONSELHO DE ENSINO, PESQUISA E EXTENSÃO APROVOU E EU,

REITOR EM EXERCÍCIO, SANCIONO A SEGUINTE RESOLUÇÃO:

 

Artigo 1° - Fica alterado o Fluxograma do Currículo do Curso de Administração, como segue:

-Exclusão dos pré-requisitos das disciplinas:

01. 40307-Direito I para 40308-Direito II;

02. 20544-Matemática CE I para 20546-Matemática Financeira;

03. 20545-Matemática CE II para 20547-Estatística;

04. 20545-Matemática CE II para 40103-Análise Micro Econômica I;

05. 40102-Economia II para 40103-Análise Micro Econômica I;

06. 40502-Contabilidade Geral II para 40504-Contabilidade de Custos I;

07. 40308-Direito II para 40310-Legislação Tributária;

08. 40502-Contabilidade Geral II para 40509-Estrutura e Análise de Balanços;

09. 40502-Contabilidade Geral II para 40204-Administração Financeira e Orçamentos I;

10. 11515-Sociologia I para 11516-Sociologia II;

11. 40202-Administração II para 40212-Administração de Materiais;

12. 40202-Administração II para 40208-Administração de Produção;

13. 40202-Administração II para 40214-Administração Pública;

14. 40202-Administração II para 40216-Administração Rural;

15. 40308-Direito II para 40311-Direito Administrativo;

16. 11601-Psicologia Aplicada à Administração para 40215-Chefia de Liderança

17. 11372-Técnico de Comunicação para 40215-Chefia Liderança;

- Inclusão dos pré-requisitos das disciplinas:

01. 20544-Matemática CE I para 40103-Análise Micro Econômica I

02. 40101-Economia I para 40103-Análise Micro Econômica I

Artigo 2º - Esta Resolução entra em vigor nesta data, revogadas as disposições

em contrário.

Dê-se ciência.

Cumpra-se.

 

 

Maringá, 29 de Novembro de 1985.

 

 

Daniel Damaszak

REITOR EM EXECÍCIO
